# Quick Reference

## Wake words

Say any of these to activate voice control:

| Wake word | Example |
|-----------|---------|
| system command | "system command, open firefox" |
| cmd | "cmd, volume up" |
| computer | "computer, search for cats" |
| i cast | "i cast, type hello world" |
| highcast | "highcast, execute system update" |

## Top commands

| Say | What happens |
|-----|--------------|
| open firefox | Opens Firefox |
| open terminal | Opens a terminal |
| volume up / volume down | Adjusts audio |
| volume mute | Toggles mute |
| search for \<query\> | Searches the web |
| type \<text\> | Types text into the focused window |
| clipboard copy / clipboard paste | Copies or pastes from clipboard |
| quit firefox | Closes Firefox |
| execute system update | Runs a system update |
| cancel | Aborts the current command |

## Modifiers

Add these words anywhere in your command:

| Word | Effect |
|------|--------|
| administrator | Runs with sudo (e.g., "administrator open firefox") |
| background | Runs in background (e.g., "open firefox background") |
| silent | Suppresses output |
| detach | Fully detaches from terminal |

## Stop the daemon

Press **Ctrl+C** in the terminal where the daemon is running, or send it a termination signal.
